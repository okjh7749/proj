Python(opencv)를 활용한 나만의 그림그리기 프로그램

-구성 구현 풀스택

1. 나의 웹캠을 활용해서 원하는 색을 추출
2. 해당 색상을 가진 물체를 영상에서 분별
3. 해당 색상의 가장큰 물체를 찾아 트래킹
4. 이후 해당 물체의 중심부를 통해 추출한 색상으로 웹캠위에 그림을 그려줌
5. 마우스를 이용한 그림그리가, 지우기 또한 지원
